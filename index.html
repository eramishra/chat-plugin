Four simple steps to use this plugin.

  1. Include js file, add following within head tags of your markup, or between body close and html close tags

    < script src= "js/chatPlugin.js" > < / script >

    Do not forget to include jQuery.js as a prerequisite to run this plugin. Add following before the js file under head tag

    < script src= "http://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js" > < / script >

  2. Include css file, add following within head tags of your markup

    < link rel= "stylesheet" href= "css/chatPlugin.js" / >

  3. Add following markup somewhere in within the body tags

    < div id= "am_container" > < / div >

  4. Finally initiate the plugin js by adding the following function, after document.ready preferably

    initiate_chat () ;
